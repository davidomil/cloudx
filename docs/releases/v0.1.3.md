# Cloudx v0.1.3

Cloudx v0.1.3 is a hotfix for the create-window dialog. It aligns the
"Create directory if needed" control with the existing create-tab dialog so
both paths use the same compact checkbox row on desktop and mobile.

## Fixes

- Replaced the create-window settings-style toggle with the shared
  `checkbox-row` treatment used by the create-tab dialog.
- Kept the mobile window form text inputs at the larger touch target size
  without stretching checkbox inputs.
- Added focused coverage for the create-window checkbox rendering and submitted
  `createDirectory` value.

## Validation

- Focused WindowSwitcher Vitest coverage passes.
- Desktop and mobile Playwright render checks pass for the create-window
  checkbox class, text, dimensions, and layout.
- Full Vitest suite passes.
- TypeScript project build passes.
- Production web/server build passes.
- Git whitespace checks pass.

## Install

```bash
git clone https://github.com/davidomil/cloudx
cd cloudx
./install.sh
```

Cloudx remains private by default and binds to `127.0.0.1`. Use
`./install.sh --lan` only for a trusted LAN or tailnet. Public internet
exposure is unsupported.
